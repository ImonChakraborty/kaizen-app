## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Expo CLI (`npm install -g @expo/cli`)

### Installation
```bash
git clone https://github.com/serene-brew/kaizen-app.git
cd kaizen-app
npm install
npx expo start -c
```

## 🛠️ Tech Stack
